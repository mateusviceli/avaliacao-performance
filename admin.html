      Guia de implementação — passo a passo
      Como colocar este protótipo no ar do zero, sem custo

      Crie sua conta no Supabase Gratuito

      Acesse supabase.com e crie uma conta gratuita. Depois clique em "New Project", escolha um nome e uma senha para o banco. Aguarde ~2 minutos enquanto o banco é criado.

      Crie a tabela de avaliações

      No Supabase, vá em SQL Editor e cole o código abaixo. Clique em "Run" para criar a tabela.

      -- Tabela principal de avaliações CREATE TABLE avaliacoes ( id UUID PRIMARY KEY DEFAULT gen_random_uuid(), criado_em TIMESTAMPTZ DEFAULT now(), data DATE, ciclo TEXT, time TEXT, colaborador TEXT, funcao TEXT, cliente TEXT, avaliador TEXT, competencias JSONB, media_competencias NUMERIC(3,2), avaliacao_geral JSONB, media_geral NUMERIC(3,2), avaliacao_colaborador JSONB, media_colaborador NUMERIC(3,2), avaliacao_cliente JSONB, media_cliente NUMERIC(3,2), media_total NUMERIC(3,2), pontos_positivos TEXT, pontos_desenvolver TEXT, objetivos JSONB, pode_treinar TEXT, ferramentas_treino TEXT, treinamentos_recebidos TEXT, modo_feedback TEXT, comentarios TEXT ); -- Permite leitura e escrita sem login (protótipo) ALTER TABLE avaliacoes ENABLE ROW LEVEL SECURITY; CREATE POLICY "acesso_publico" ON avaliacoes FOR ALL USING (true) WITH CHECK (true);
      3

      Copie suas credenciais do Supabase

      No painel do Supabase, vá em Settings → API. Copie o Project URL e a anon/public key. Cole nos dois arquivos HTML (index.html e dashboard.html) nas linhas marcadas com SUPABASE_URL e SUPABASE_KEY.

      4

      Suba os arquivos no GitHub Gratuito

      Crie uma conta em github.com. Crie um repositório novo (pode ser privado). Faça upload dos 3 arquivos HTML: index.html, dashboard.html e admin.html.

      5

      Ative o GitHub Pages Gratuito

      No repositório GitHub, vá em Settings → Pages. Em "Source" selecione main branch e clique em Save. Em ~1 minuto seu sistema estará disponível em seuusuario.github.io/nome-do-repo. Compartilhe esse link com o time.

      6

      Pronto — teste o sistema

      Acesse o link do GitHub Pages, preencha uma avaliação de teste e verifique no Supabase (aba Table Editor → avaliacoes) se o registro apareceu. Depois acesse /dashboard.html e veja os indicadores.

      ⚠️ Lembre-se: o Supabase Free pausa o projeto após 7 dias sem acesso. Quando os custos forem aprovados, basta migrar para o plano Pro (um clique) e todos os dados são mantidos.
